# AGENTS.md

## 1. Project Overview

이 프로젝트는 **ICON Lab 연구 내용을 소개하는 연구실 웹사이트**를 구축하기 위한 프로젝트이다.

웹사이트는 다음 두 자료를 기반으로 제작된다.

Reference style:
- https://github.com/97yong/yongchae

Reference content:
- https://sites.google.com/view/icon-lab

목표는 Google Sites 기반 페이지를 **정적 웹사이트(HTML + CSS + JS)** 로 재구성하는 것이다.

---

# 2. Primary Goals

본 프로젝트의 주요 목표는 다음과 같다.

1. Google Sites 기반 연구실 페이지를 **독립적인 웹사이트로 재구성**
2. **연구실 소개 / 연구 분야 / 논문 / 구성원** 정보를 체계적으로 제공
3. **한글을 기본 언어로 제공하고 영어 페이지도 지원**
4. **GitHub Pages 배포 가능 구조로 구현**
5. **모바일 친화적 UI 제공**

---

# 3. Language Policy

웹사이트는 **한국어를 기본 언어**로 사용한다.

그러나 국제 방문자를 위해 **영문 페이지도 제공한다.**
